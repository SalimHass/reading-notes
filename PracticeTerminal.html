what is a command line :A command line, or terminal, is a text based interface to the system. You are able to enter commands by typing them on the keyboard and feedback will be given to you similarly as text.

some of the most important commands

pwd: it will print your currnet directory path- ie. Where are you currently. ls: it will list the content of the currnet directory , both folders and files with the extintions. cd: Change Directories , if followed by ".." you will take one step up in the path , if followed by a folder name it will enter that folder if exist where you currntly are, if used alone , it will jump to the root . note that if you typed a path after it it will enter that path. mkdir -newdir-: to make a new directory and will name it newdir rmdir -newdir-: will remove the newdir directory touch -newfl-: will creat a new file and will name it newfl cp -name- -source- -destination-: will copy the file or folder -name-, from the source path to the destination. mv -name- -source- -destination-: will move the file or folder -name-, from the source path to the destination.
note that there is no undo in Linux

Files

Linux is an Extensionless System and case sensitive, Under Linux the system actually ignores the extension and looks inside the file to determine what type of file it is, Also because directories (as mentioned above) are actually just a special type of file.

Use Quotes for file name that has spaces If the file or directory's name begins with a . (full stop) then it is considered to be hidden “ls -a Documents” to view hidden file

Manual Pages

The manual pages are a set of pages that explain every command available on your system including what they do, the specifics of how you run them and what command line arguments they accept.
man <command to look up>
man -k <search term> to do a keyword search on the Manual pages
