Reading and Writing Files in Python

a file is a contiguous set of bytes used to store data, its consist of three parts:

Header:metadata about the contents of the file (file name, size, type, and so on)

Data: contents of the file as written by the creator or editor

End of file(EOF): special character that indicates the end of the file

the file path consists of three parts as well , Folder path which is the location of the file ,

File Name which is the actual name of the file , Extention which is the end of the file pre-apended with a dot (.) its used indicate the fiel type.

to access a path , you can write the whole path or relative path , use (./file) if it is in the same level , or (../file) if its up by one level,

(../../file) means two levels away

The try and except Block: Handling Exceptions

The try and except block in Python is used to catch and handle exceptions. Python executes code following the try statement as a “normal” part of the program.

The code that follows the except statement is the program’s response to any exceptions in the preceding try clause.
